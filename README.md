Big Mart Sales Prediction

Description:
This project predicts sales of products in Big Mart using Python and XGBoost. The notebook covers data cleaning, preprocessing, exploratory data analysis (EDA), feature engineering, and regression modeling.

Features

Handled missing values (Item_Weight, Outlet_Size)

Standardized categorical data (Item_Fat_Content)

Explored numerical and categorical features with visualizations
